Instructions
CSV Format Requirements:
  • File must be semicolon-separated (;)
  • First row must contain column headers
  • Required columns:
    • product_number
    • description
    • supplier_code
    • category_name
    • color_name
    • material_name
  • Optional picture columns (up to 5 pictures):
    • picture_1_name, picture_1_url, picture_1_order
    • picture_2_name, picture_2_url, picture_2_order
    • etc...
Notes:
  • If a component with the same product_number exists, it will be updated
  • New suppliers, categories, colors, and materials will be created automatically if they don't exist
  • For updates, all pictures will be replaced with the ones in the CSV
  • Empty picture fields will be ignored
